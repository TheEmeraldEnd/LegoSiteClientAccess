# LegoSiteClientAccess
Used to access the LegoTestSite API when both are running. This one just formats it into the browser.

Important pieces of the project:

Program:
	Utilized as a jumpoff point. This program requires the LegoTestSite side to be running before this is.
(will set up a version later in the future that doesn't require this dependancy to test).

SetDetails.razor:
	The razor page used to show the set details of what lego pieces go into which lego bags. 

SetGallery.razor:
	Used to show which sets are currently catalogued.
